# RPG Manager

A small Python practice project created while refreshing Python fundamentals and learning object-oriented programming. The project is being developed incrementally to explore classes, JSON data storage, inventory systems, and general software design principles.

## Features

- Create custom characters
- Save and load character data using JSON
- Turn-based combat against a training dummy
- Character status display
- Health and healing system
- Inventory system with item management
- Input validation with exception handling
- Interactive CLI menus
